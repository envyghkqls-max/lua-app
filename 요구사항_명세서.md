# 요구사항 명세서

**프로젝트명:** 중학생을 위한 스터디 & 시험/수행평가 관리 앱  
**플랫폼:** 앱인토스 (Toss 미니앱)  
**작성일:** 2026-06-18  
**상태:** 초안 (인터뷰 기반)

---

## 1. 프로젝트 개요

### 목적
중학생이 시험 및 수행평가 일정을 놓치지 않도록 관리하고, 공부 계획 및 성적 추이를 한 곳에서 확인할 수 있는 앱인토스 기반 미니앱.

### 타겟 사용자
- 중학교 1~3학년 학생
- 스마트폰(토스 앱)을 통해 학습 일정을 관리하고 싶은 사용자

---

## 2. 핵심 기능 (MVP)

### 2-1. 시험 / 수행평가 일정 등록

| 입력 항목 | 내용 |
|-----------|------|
| 과목명 | 국어, 수학, 영어 등 |
| 날짜 | 시험/수행평가 날짜 선택 |
| 유형 | 시험 / 수행평가 구분 |
| 시험 범위 | 단원(Unit) 기반 입력 (ex. 3단원 ~ 5단원) |
| 수행평가 유형 | 별무덕해, 프레젠테이션, 도서연시 등 직접 입력 |
| 비율 & 배점 | 반영 비율(%), 만점 기준 (ex. 30점 만점) |

### 2-2. D-day 알림

- 사용자가 일정별로 알림 시점을 직접 설정 (예: 7일 전, 3일 전, 당일 등)
- 기기 푸시 알림 or 토스 내 알림 활용

### 2-3. 홈 화면 (대시보드)

- **D-day 리스트 집중 뷰**: 다가오는 시험/수행평가를 D-day 순으로 정렬 & 카드 형태로 표시
- 각 카드에: 과목명, D-day 카운트다운, 유형 배지(시험/수행평가) 표시

### 2-4. 공부 계획 (간단 메모)

- 일정마다 한 줄 메모 형태의 공부 계획 입력 가능
- 복잡한 Task 관리보다는 빠르게 적는 수준

### 2-5. 성적 기록 & 추이 그래프

- 시험 후 점수 입력 기능
- 과목별 점수 추이를 선 그래프로 시각화
- 수행평가 점수도 별도 기록 가능

### 2-6. 게이미피케이션

- 공부 계획 달성 시 보상 요소 (뱃지, 마스코트 반응 등)
- 연속 구부구부(스트릭) 기능 — 연속으로 계획을 달성하면 시각적으로 표시
- 중학생의 동기 부여를 위한 가벼운 성취 시스템

## 3. 디자인 & UX

### 테마
- 사용자가 앱 내에서 컬러 테마를 직접 선택 가능
- 추천 테마 후보: 클린 화이트, 다크모드, 파스텔 핑크, 민트 등

### UX 원칙
- 토스 미니앱 디자인 가이드라인 준수
- 중학생 대상이므로 직관적이고 빠른 입력 UX 우선
- 복잡한 설정보다 "바로 쓸 수 있는" 경험 지향

## 5. 로드맵 (MVP 이후)

| 기능 | 설명 | 우선순위 |
|------|------|----------|
| 반 친구와 일정 공유 | 같은 반 친구들과 수험 일정 / 공부 계획 공유 | 중 |
| AI 시험 범위 요약 | 교과서 사진 촬영 → AI가 시험 범위 자동 정리 | 높음 |
| 공부 타이머 (Pomodoro) | 일정에 연결된 뒤 집중 타이머 제공 | 높음 |
| 원소 형포지 스케줄링 | 국리원, 수학, 영어 등 시간표 자동 스케줄링 | 낮음 |

## 6. 기술 고려사항 (초안)

- **플랫폼:** 앱인토스 (Toss Super App 미니앱 SDK)
- **데이터 저장:** 사용자별 로컬 or 토스 계정 연동 서버 저장
- **알림:** 기기 푸시 알림 or 토스 인앱 알림
- **그래프:** 경량 차트 라이브러리 (예: Recharts, Chart.js 등)
